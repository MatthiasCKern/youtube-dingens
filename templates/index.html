Wöchentliche YouTube Analyse

So funktioniert es:

  1. Gehe zu Google Takeout.
  2. Wähle nur YouTube und YouTube Music aus
  3. Klicke auf Mehrere Formate und wähle bei Verlauf das Format JSON
  4. Klicke auf Weiter, wähle ZIP und z. B. 2 GB, und erstelle den Export
  5. Du erhältst per E-Mail einen Link zum Download der ZIP-Datei.
  6. Lade die json Datei hier im Tool hoch, um deinen Verlauf auszuwerten.
